## Some simple Cypress examples

**To setup project**

May not be needed

``` npm init -y ```

Needed

``` install npm ```

 ``` npm install cypress --save-dev ```


**Run tests**

Headless
 ``` npx cypress run ```
 
 Browser
 ``` npx cypress open ```

*Okay great, lets go!!!!*

Also don't forget to update your ApliTools API key, steps here:
https://applitools.com/tutorials/getting-started/setting-up-your-environment.html
